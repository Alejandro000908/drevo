## 🎨 DISEÑO - MANTENIENDO ELEGANCIA

### Elementos utilizados:
- **Bloques de contenido**: bg-gray-50 dark:bg-gray-900 rounded-2xl p-8
- **Listas visuales**: Con checkmarks (✓) en color primario
- **Cards informativas**: Grid de 2-3 columnas
- **FAQ**: `<details>` accordion nativo HTML
- **CTA sections**: Gradientes de marca (from-[#009479] to-[#00BFA5])
- **Spacing**: py-20 entre secciones, mb-12 para headings

### NO hacer:
- ✗ Bloques de texto largos sin formatear
- ✗ Muros de texto sin imágenes/whitespace
- ✗ Keyword stuffing visible
- ✗ Diseño tipo blog/artículo

### Template base:
```tsx
import Header from '@/components/Header'
import Footer from '@/components/Footer'
import Link from 'next/link'

export const metadata = {
  title: '[Título SEO con keyword]',
  description: '[150-160 chars con keywords naturales]',
  keywords: ['keyword1', 'keyword2', ...],
  alternates: { canonical: `${siteConfig.url}/ruta/` },
}

export default function Page() {
  return (
    <>
      <Header />
      <main className="min-h-screen pt-20">
        {/* Hero con H1 */}

      <Footer />
    </>
  )
}
```
